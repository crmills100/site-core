The workflow

From GP pipeline to committed investor

Every step of the private equity onboarding journey — from first contact to executed subscription agreement and confirmed position — in a single platform.

1
Build your pipeline
Import your investor CRM via CSV or add contacts manually. Track every prospect through a visual Kanban — Prospect, Interested, Data Room, Subscription, Committed.
2
Grant data room access
Upload deal materials to a bank-grade encrypted data room. Control access at the document level — view only, view & download, or GP-internal. NDA gating built in.
3
Send subscription docs
Generate and deliver subscription agreements via DocuSign. Gated by data room completion. Track signature status in real time. Automated reminders built in.
4
Accept & create positions
Once both parties have signed, accept the executed agreement within the platform to formally create the investor's position — units, unit price, effective date, and all terms confirmed.
